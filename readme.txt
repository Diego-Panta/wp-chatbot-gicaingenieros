== Descripción ==
Este plugin agrega un chat flotante para consultas en WordPress.

== Instalación ==
1. Sube el plugin a la carpeta `/wp-content/plugins/`.
2. Actívalo desde el panel de administración de WordPress.
3. ¡Listo! El chat flotante aparecerá en tu sitio.
